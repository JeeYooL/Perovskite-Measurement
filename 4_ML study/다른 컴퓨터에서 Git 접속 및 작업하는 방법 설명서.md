# 💻 다른 컴퓨터에서 코드를 다운받아서 수정하고 다시 올리는 방법 (Git & GitHub 상세 가이드)

이 설명서는 연구실, 집, 또는 노트북 등 **서로 다른 컴퓨터를 오가며** 하나의 코드를 수정하고 안전하게 동기화하기 위한 아주 구체적인 사용법입니다.

---

## 0️⃣ 시작하기 전에: 다른 컴퓨터 최초 1회 세팅

다른 컴퓨터(집 데스크탑, 노트북 등)에서 처음 작업을 시작할 때 '단 한 번'만 해주는 설정입니다.

1. **Git 설치하기** 
   - [Git 다운로드 페이지(클릭)](https://git-scm.com/download/win)에 들어가서 `64-bit Git for Windows Setup`을 다운로드하고 설치합니다. (수많은 옵션이 나오지만 전부 기본값 그대로 두고 `Next`만 눌러 설치 완료하시면 됩니다.)

2. **내 코드 전체 다운로드 받기 (Clone)**
   - 코드를 저장하고 싶은 빈 폴더나 바탕화면 빈 공간에서 마우스 오른쪽 버튼을 클릭하고 **`Open Git Bash here`** (또는 터미널 열기)를 선택합니다.
   - 열린 검은 창(터미널)에 아래 코드를 복사해서 붙여넣고 엔터를 치세요!
   ```bash
   git clone https://github.com/JeeYooL/Perovskite-Measurement.git
   ```
   *👉 이제 내 컴퓨터에 `Perovskite-Measurement`라는 폴더가 생겼습니다! 이 안으로 들어가서 편집기(VS Code, 커서, 파이참 등)를 실행해 주시면 세팅 끝입니다.*

---

## 🚀 1️⃣ 작업 시작 직전: 코드 최신화 (가장 중요!)

오늘 다른 컴퓨터에서 작업을 시작하기 전에, **가장 먼저 필수로 쳐야 하는 1번 명령어**입니다. (만약 연구실에서 수정한 코드가 있다면 그걸 이 컴퓨터로 끌고 와야 코드 충돌이 안 생깁니다.)

편집기의 터미널 창을 열고 아래 명령어를 칩니다.
```bash
git pull
```
- 성공했다면, 코드가 촤르륵 다운받아지거나 `Already up to date`(이미 최신 상태입니다)라는 문구가 뜹니다.

---

## ✍️ 2️⃣ 평소처럼 재밌게 코딩하기

최신 코드를 받아왔다면 이제 평소처럼 자유롭게 파이썬(`.py`) 코드를 만들거나 기존 코드를 편하게 수정하시면 됩니다! 
- 코드 실행해보기
- 에러 고치기
- (Ctrl + S 를 눌러서 파일들 저장 잊지 마세요!)

---

## 💾 3️⃣ 퇴근(작업 종료) 전: 수정한 코드를 허공에 날리지 않고 GitHub에 올려두기

코딩을 얼추 다 끝내고 "이제 다른 컴퓨터 가야지~" 혹은 "이제 컴퓨터 꺼야지!" 할 때, 방금 **열심히 작성하신 코드를 아주 안전하게 클라우드(GitHub)에 박제시키는 삼단 콤보 필수 명령어**입니다.

**1. 변경된 모든 파일 한 번에 장바구니 담기**
```bash
git add .
```
*(※ 주의: `add`와 `.`(마침표) 사이에는 무조건 스페이스바 한 칸이 있어야 합니다!)*

**2. 장바구니에 오늘 내가 무슨 작업을 했는지 포스트잇 메모장 붙이기**
```bash
git commit -m "오늘의 작업 요약 메모"
```
*(예시: `git commit -m "ML 데이터 전처리 기초 코드 연습함"`, `git commit -m "측정 코드 에러나는 부분 수정"`. 나중에 회원님이 다시 봤을 때 알아볼 수 있는 아무 말이나 적어주세요!)*

**3. 실제로 발송 버튼 누르기! (클라우드 업로드)**
```bash
git push
```
*(터미널 창에 100% 완료되었다는 문구가 뜨면 모두 완벽하게 성공한 것입니다. 이 명령어를 쳐야 진짜로 백업이 완료됩니다.)*

## 💡 마무리 초간단 요약 가이드 (이거 하나면 끝!)

정말 헷갈린다면 이것만 기억하시면 됩니다!

1. 컴퓨터 켜고 맨 처음: `git pull`
2. 열심히 코딩: 👨‍💻
3. 컴퓨터 끄기 전 맨 마지막: `git add .` -> `git commit -m "메모"` -> `git push`
